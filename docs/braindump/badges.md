# Badges

It would be nice if users could earn some sort of progression related badges, resulting in a bit of gamification within the more scientific approach of this application.

This can be related to certain categories, such as:

- Dedication / consequtive training
- Total darts thrown
- 180's (1st, 10th, 100th etc.)
- Average progression (for example every full point)
- Best leg
- Highest average
- Wins
- Establishing favourite double
