## Identity

One of the classic darts games: race from a starting score down to exactly zero. Standard dartboard scoring is assumed. Sibling start scores (301, 701) and alternate in/out rules exist as later variants.

## Objective

- **Leg:** reduce your score from the start total to **exactly 0**, finishing on a **double** (double out, including double bull).
- **Session (V1):** **first to N** legs — win N legs to finish the session. Each new leg starts fresh from the start score.

## How to play (V1)

### Visit

A **visit** is one turn at the board: the player may throw **up to three darts**. After those darts, the visit ends and play passes to the next player. In single-player (V1), that still means one visit at a time — finish the visit, then start the next.

If the leg is won on the first or second dart (a successful checkout), the visit **ends immediately** — the remaining darts are not thrown.

### Progress

Each scoring dart subtracts from the remaining total. Play continues visit by visit until the player checks out or busts.

### Finishing

Under V1 **double out**, the dart that reaches exactly 0 must be a double (including double bull). That wins the leg.

### Bust

If a visit would take the player past 0, or leave a score that cannot be finished under the active out rule (for example, remaining on 1 when the game must end on a double), that visit is a **bust**. The darts in that visit do not count; the player’s score goes back to what it was at the start of the visit, and the turn is over.

## Later versions (V2+)

### Variants

- **Start score:** other X01 values such as **301** and **701** (same rules as 501, different starting total)
- Alternate **in / out** modes — defined in the Glossary; **not selectable in V1**

### Match structure

- **Best of N legs** — introduced with multiplayer (not used in V1 single-player)
- First to N sets / best of N sets
- Sets of 3 or 5 legs
- Optional rule: winner must win by a margin of at least 2 legs
- Same margin rule can apply to the last set when players are level on sets and the set reaches 1-1 or 2-2 in legs

## Glossary

| Term                    | Version | Meaning                                                                                                 |
| ----------------------- | ------- | ------------------------------------------------------------------------------------------------------- |
| **Open in**             | V1      | Every dart counts from the start of the leg.                                                            |
| **Double out**          | V1      | The finishing dart must be a double (including double bull).                                            |
| **Double in**           | V2+     | The leg does not start counting until the player hits a double. Darts before that double score nothing. |
| **Master in**           | V2+     | Same idea as double in, but the opening hit must be a **master**: a double, a triple, or the bull.      |
| **Master out**          | V2+     | The finishing dart must be a master: a double, a triple, or the bull.                                   |
| **Straight / open out** | V2+     | The leg can be finished on any dart that reaches exactly 0 (no double/master required).                 |
| **First to N**          | V1      | Session ends when one side has won N legs.                                                              |
| **Best of N**           | V2+     | Match length format for multiplayer; not used in V1.                                                    |
