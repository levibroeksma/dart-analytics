## Identity

Checkout and counting practice built on X01-style scoring. Start at **121**, try to finish to 0 within a limited dart budget; success climbs the target (122, 123, …). Standard dartboard scoring is assumed; finishing is double out.

## Objective

- **Attempt:** from the current target score, reach **exactly 0** on a **double** within the allowed darts (3, 6, or 9).
- **Session (V1):** keep climbing after each success. Failure handling depends on difficulty (see config / later versions).

## How to play (V1)

### Visit

A **visit** is up to **three darts**. The dart budget is a set number of visits (e.g. 9 darts = three visits). Scoring works like X01: subtract each dart from the remaining total.

### Progress

Begin at **121**. Within the dart budget, try to check out. On success, the next attempt starts at **current + 1** (122, then 123, and so on).

### Finishing

Must reach exactly 0 on a double (including double bull), same idea as 501 double out.

### Bust

If a visit would go past 0, leave 1 under double out, or hit 0 without a double, that visit is a **bust**: those darts do not count; score returns to the start of the visit. Remaining visits in the budget (if any) continue from that restored score.

### Failure (end of budget)

If the checkout is not made within the dart budget, apply the fail rule. V1 easy: **stay** on the current target and try again.

## Later versions (V2+)

### Variants

- Dart budget selectable: **3 | 6 | 9**
- Fail rules:
  - **Easy:** stay on current number
  - **Hard:** move back **1**
  - **Extreme:** reset to **121**
- Optional **safehouse**: a 3-dart checkout locks a floor so you cannot drop below that score later
- Optional climb cap / win when a chosen end target is checked out

### Match structure

- Multiplayer: players alternate attempts; race or shared ladder as product defines

## Glossary

| Term             | Version  | Meaning                                                                |
| ---------------- | -------- | ---------------------------------------------------------------------- |
| **Dart budget**  | V1 / V2+ | How many darts allowed per attempt at the current target (3, 6, or 9). |
| **Easy fail**    | V1       | Miss the budget → stay on the same target.                             |
| **Hard fail**    | V2+      | Miss the budget → target decreases by 1.                               |
| **Extreme fail** | V2+      | Miss the budget → target resets to 121.                                |
| **Safehouse**    | V2+      | Optional: a 3-dart checkout sets a floor you cannot fall below.        |
